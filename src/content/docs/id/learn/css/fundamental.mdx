## What is CSS?

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

## How to Apply CSS

There are 3 ways to apply CSS to HTML

### Inline CSS

Write directly css inside an HTML element using the `style` attribute.

```html "style="color: red; font-size: 20px;""
<p style="color: red; font-size: 20px;">This is an inline-styled paragraph.</p>
```

### Internal CSS
Wrap css with style tag in `<head>` section.
```html ins={2-7}
<head>
  <style>
    p {
      color: blue;
      font-size: 18px;
    }
  </style>
</head>
<body>
  <p>This text should blue.</p>
</body>
```

### External CSS

Similar to internal css but the CSS code is written in a file

```html ins={2}
<head>
  <link rel="stylesheet" href="styles.css">
</head>
```

## Selectors

CSS selector is used to select the HTML element for styling.

```html
<html>
  <head>
    <style>
      /* Universal Selector */
      * {
        margin: 0;
        padding: 0;
      }

      /* Element Selector */
      p {
        color: red;
      }

      /* Class Selector */
      .content {
        background-color: blue;
      }

      /* ID Selector */
      #main-heading {
        color: white;
      }
    </style>
  </head>
  <body>
    <article class="content">
      <h1 id="main-heading">White Color</h1>
      <p>this background-color should be blue</p>
      <p>this text should be red</p>
    </article>
  </body>
</html>
```
