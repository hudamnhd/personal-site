## 1. Introduction to CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML. CSS controls the layout, design, and visual appearance of web pages, including aspects like colors, fonts, spacing, and positioning.

### Key Points:
- CSS is used to style HTML elements.
- It separates the content (HTML) from the design (CSS), making web development more modular.
- CSS can be applied inline, internally, or externally.

## 2. How to Apply CSS

There are three main ways to apply CSS to a web page:

1. **Inline CSS**: Directly inside an HTML element using the `style` attribute.
2. **Internal CSS**: Inside a `<style>` tag in the `<head>` section of the HTML document.
3. **External CSS**: In a separate `.css` file linked to the HTML document using the `<link>` tag.

### Example: Inline CSS
```html
<p style="color: red; font-size: 20px;">This is an inline-styled paragraph.</p>
```

### Example: Internal CSS
```html
<head>
  <style>
    p {
      color: blue;
      font-size: 18px;
    }
  </style>
</head>
<body>
  <p>This is an internally-styled paragraph.</p>
</body>
```

### Example: External CSS
```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
```

## 3. Selectors

CSS selectors are used to target HTML elements for styling. There are several types of selectors.

### Types of Selectors:

- **Element Selector**: Targets all instances of a specific HTML element.
```css
p {
  color: green;
}
```

- **Class Selector**: Targets elements with a specific class attribute.
```css
.button {
  background-color: blue;
}
```

- **ID Selector**: Targets an element with a specific id attribute.
```css
#header {
  font-size: 24px;
}
```

- **Universal Selector**: Targets all elements on the page.
```css
* {
  margin: 0;
  padding: 0;
}
```

- **Attribute Selector**: Targets elements with a specific attribute.
```css
a[href^="https"] {
  color: green;
}
```

## 4. Colors in CSS

Colors can be defined in CSS in several ways, including color names, hex values, RGB, RGBA, HSL, and HSLA.

### Examples:
```css
color: red; /* Named color */
color: #ff0000; /* Hexadecimal color */
color: rgb(255, 0, 0); /* RGB color */
color: rgba(255, 0, 0, 0.5); /* RGBA (with transparency) */
color: hsl(0, 100%, 50%); /* HSL color */
color: hsla(0, 100%, 50%, 0.5); /* HSLA (with transparency) */
```

## 5. Box Model

The box model in CSS represents the structure of an element on the page. Each element is represented as a rectangular box consisting of the following parts:

- **Content**: The actual content of the element (text, images, etc.)
- **Padding**: Space between the content and the border.
- **Border**: The border surrounding the padding and content.
- **Margin**: Space outside the border, separating the element from others.

### Example:
```css
div {
  width: 200px;
  height: 100px;
  padding: 10px;
  border: 5px solid black;
  margin: 20px;
}
```

## 6. Layout in CSS

CSS provides several methods for controlling the layout of elements on a page:

### Flexbox

Flexbox is a layout model that allows you to design complex layouts with ease.

#### Example:
```css
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}
```

### Grid

CSS Grid allows you to create grid-based layouts with rows and columns.

#### Example:
```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
```

### Positioning

You can control the position of elements using various position properties:

- **static**: The default value. Elements are positioned according to the normal document flow.
- **relative**: Positioned relative to its normal position.
- **absolute**: Positioned relative to the nearest positioned ancestor.
- **fixed**: Positioned relative to the viewport.
- **sticky**: Positioned based on the user's scroll position.

#### Example:
```css
.container {
  position: relative;
  top: 10px;
  left: 20px;
}
```

## 7. Fonts and Typography

CSS allows you to customize text appearance, including font family, size, weight, and style.

### Example:
```css
p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  line-height: 1.5;
}
```

### Google Fonts
To use Google Fonts, include the font link in the `<head>` section and apply the font-family in your CSS.

```html
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap" rel="stylesheet">
```

```css
body {
  font-family: 'Roboto', sans-serif;
}
```

## 8. Responsive Web Design

Responsive web design allows your website to adjust to different screen sizes and devices. This is typically achieved using CSS media queries.

### Example:
```css
@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}
```

### Key Points:
- Media queries allow you to apply styles based on the viewport's width, height, and other features.
- Common breakpoints: 320px (mobile), 768px (tablet), 1024px (desktop).

## 9. Transitions and Animations

CSS allows you to add animations and smooth transitions between property changes.

### Transition Example:
```css
button {
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: green;
}
```

### Animation Example:
```css
@keyframes slide {
  from {
    transform: translateX(0);
  }
  to {
    transform: translateX(100px);
  }
}

div {
  animation: slide 2s infinite;
}
```

## 10. Conclusion

CSS is essential for styling and laying out web pages. Understanding the fundamentals of CSS, such as selectors, colors, box model, layout techniques, and responsiveness, will help you create visually appealing and functional websites. CSS is continually evolving with new features, so it's important to stay updated with best practices.

Happy coding!
